Ashby Docs MCP

Give your AI assistant instant access to all 215 Ashby help articles.

What this does: This adds a tool to your AI assistant (Claude or Cursor) that lets it search and read Ashby's help documentation directly. Instead of guessing or sending you to Google, your AI can look up the actual answer from Ashby's knowledge base — things like how approvals work, how to configure automations, how to set up job postings, and more.

How to install: Find your AI tool below, follow the steps to open the right config file, paste in the code snippet, save, and restart. That's it.

Claude Code (terminal)

If you use Claude Code in your terminal (the claude command), follow these steps:

  1. Open your terminal and run: open ~/.claude.json
  2. Find the section that says "mcpServers"
  3. Paste the code below inside that section (after the opening {, before the closing })
  4. Make sure there's a comma separating it from any other entries
  5. Save the file, then start a new Claude session

~/.claude.json → under "mcpServers"

"ashby-docs": {
  "type": "http",
  "url": "https://ashby-docs-mcp.netlify.app/mcp"
}

Not sure where to paste it? Ask Claude to add it for you — just say "add this to my mcpServers in ~/.claude.json" and paste the snippet.

Claude Desktop

If you use the Claude desktop app on Mac, follow these steps:

  1. Open Finder, press Cmd + Shift + G, and paste this path: ~/Library/Application Support/Claude/
  2. Open the file called claude_desktop_config.json in a text editor
  3. Find the section that says "mcpServers"
  4. Paste the code below inside that section
  5. Make sure there's a comma separating it from any other entries
  6. Save the file, then fully quit and reopen Claude Desktop

~/Library/Application Support/Claude/claude_desktop_config.json → under "mcpServers"

"ashby-docs": {
  "type": "http",
  "url": "https://ashby-docs-mcp.netlify.app/mcp"
}

Cursor

If you use Cursor, the easiest way is through the settings UI:

  1. Open Cursor and go to Settings → Cursor Settings → MCP
  2. Click Add new MCP server
  3. Set the type to http and paste the URL: https://ashby-docs-mcp.netlify.app/mcp
  4. Save and restart Cursor

Or if you prefer to edit the config file directly:

~/.cursor/mcp.json → under "mcpServers"

"ashby-docs": {
  "type": "http",
  "url": "https://ashby-docs-mcp.netlify.app/mcp"
}
